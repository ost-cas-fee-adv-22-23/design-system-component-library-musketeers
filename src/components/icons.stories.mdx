# Icons

Icons will be automatically generated from src/assets/icons/\*.svg via SVGR and converted to React Components.

Tailwind's "text-" class can be used to change the color of the Icon.<br/>
"width" and "height" attributes can be used to set the size in pixels:

<br />

## Example

`import Calendar from "../assets/icons/Calendar.svg";`<br/>
`...`<br/>
`<Calendar className="text-violet-600" width="50" height="50" />`

<Calendar className="text-violet-600" width="50" height="50" />
